# 📊 Retail Sales Analytics Project (SQL + Power BI)

## 🔍 Project Overview

This project focuses on analyzing retail sales performance by combining advanced SQL data transformation with an interactive Power BI dashboard. It uncovers actionable insights related to sales trends, customer behavior, and external factors such as weather conditions.

---

## 🎯 Business Objectives

* Analyze daily and store-level sales performance
* Identify key factors impacting sales (weather, demographics)
* Track KPIs and performance trends
* Enable data-driven business decisions

---

## 🛠️ Tech Stack

* **SQL (MySQL)** – Data transformation & feature engineering
* **Power BI** – Data visualization & dashboard creation
* **Excel/CSV** – Data source

---

## 🧠 Key SQL Features Used

* **Window Functions**

  * `RANK()` – Rank stores based on sales performance
  * `LAG()` – Analyze previous day sales
* **Derived Metrics**

  * Day-over-day **sales growth**
  * **7-day rolling average** for trend analysis
* **Joins**

  * Integrated Sales, Weather, and Survey datasets

---

## 📈 Key Insights

* Sales vary significantly based on **weather conditions** such as rain and temperature
* High-performing stores are consistently identified using ranking techniques
* Negative sales growth highlights potential demand fluctuations
* Rolling averages help smooth trends and reveal patterns

---

## 📊 Dashboard Features

* KPI Cards: Total Sales, Average Sales, Growth
* Sales Trend Analysis with rolling averages
* Store-wise Performance Ranking
* Weather Impact on Sales
* Customer Demographics Insights
